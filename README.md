# OAE-model-loader

## Install

* git clone git://github.com/sakaiproject/OAE-model-loader.git
* brew install node
* npm install canvas

## Add files

Add create the following folders & add image files to 

* `./data/pictures/users`
* `./data/pictures/worlds`

## Run

* node generate.js
* node loaddata.js 10 http://localhost:8080 admin 10
* node main.js
